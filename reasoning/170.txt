O produto custa R$ 150.000 à vista, mas o cliente decide financiá-lo, de forma que o valor à vista é aumentado em 20% antes de ser dividido em três parcelas iguais. Após o acréscimo de 20%, o novo valor à vista para ser dividido em parcelas é:
R$ 150.000 + (20% de R$ 150.000) = R$ 150.000 + R$ 30.000 = R$ 180.000.

Esse valor é dividido em três parcelas iguais, logo cada parcela será de:
R$ 180.000 / 3 = R$ 60.000. 

A primeira parcela é paga no ato, sem juros. As outras duas parcelas serão pagas com 30 e 60 dias de diferença, respectivamente, e estarão sujeitas a juros compostos. Vamos chamar de P o valor final de cada parcela, de i a taxa mensal de juros e de n o número de meses. Note que, como estamos considerando juros compostos mensalmente, devemos converter o número de dias para meses para os cálculos, ou seja, 30 dias é aproximadamente 1 mês e 60 dias é aproximadamente 2 meses.

A fórmula para achar o valor final de uma aplicação com juros compostos é:
P = C * (1 + i)^n.

Neste caso, cada parcela tem o valor inicial (C) de R$ 60.000. A primeira parcela paga imediatamente (n = 0) terá o valor de P = R$ 60.000 * (1 + i)^0 = R$ 60.000, pois (1 + i)^0 é igual a 1.

Para a segunda parcela (n = 1 mês), temos:
R$ 60.000 = C * (1 + i)^1 => C = R$ 60.000 / (1 + i).

Para a terceira parcela (n = 2 meses), temos:
R$ 60.000 = C * (1 + i)^2 => C = R$ 60.000 / (1 + i)^2.

Já que o valor emprestado foi de R$ 180.000, a soma das três parcelas descontadas é igual a R$ 180.000. 

Assim, temos:
C + C + C/(1+i) + C/(1+i)^2 = R$ 180.000.

Como C é o valor da primeira parcela, que é R$ 60.000, obtemos:
R$ 60.000 + R$ 60.000 / (1 + i) + R$ 60.000 / (1 + i)^2 = R$ 180.000.

Dividindo todos os termos por R$ 60.000, obtemos:
1 + 1 / (1 + i) + 1 / (1 + i)^2 = 3.
